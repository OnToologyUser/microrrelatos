A continuación se muestran una serie de ejemplos de consultas [SPARQL](https://www.w3.org/TR/rdf-sparql-query/) que se pueden lanzar sobre la ontología poblada. 1. El autor del primer microrrelato: ``` PREFIX mcr: select distinct ?autor{ mcr:microrrelato_1 mcr:esObraArtisticaCreadaPor ?autor } ``` 2. El nombre y los apellidos del autor del primer microrrelato: ``` PREFIX mcr: select distinct ?nombre ?apellidos{ mcr:microrrelato_1 mcr:esObraArtisticaCreadaPor ?autor . ?autor mcr:nombreDePila ?nombre . ?autor mcr:apellidos ?apellidos . } ``` 3. Listado de todos los títulos de los microrrelatos con sus autores: ``` PREFIX mcr: select distinct ?titulo ?nombre_autor{ ?uri_microrrelato mcr:esObraArtisticaCreadaPor ?uri_autor . ?uri_autor rdfs:label ?nombre_autor . ?uri_microrrelato rdfs:label ?titulo } ```
